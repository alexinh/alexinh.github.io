Таблицы

Тег <caption>

Предназначен для создания заголовка к таблице.

Тег <col>

Задает ширину и другие характеристики одной или нескольких колонок таблицы.

Тег <table>

Создает таблицу.

Тег <td>

Предназначен для создания одной ячейки таблицы.

Тег <th> <tr>

Создает одну ячейку таблицы, которая обозначается как заголовочная. Служит контейнером для создания строки таблицы.
